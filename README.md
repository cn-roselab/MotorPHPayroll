# MotorPH Payroll System

## 📌 Project Description
The **MotorPH Payroll System** is a Java-based application designed to process employee information and attendance records to compute payroll accurately.  

The system reads data from CSV files and calculates employee salary based on:
- Attendance records
- Working hours
- Employee details

---

## 📂 Files Included
- **MotorPHemployees.csv**  
  Contains employee information such as employee ID, name, and salary details.

- **MotorPHemployeeattendance.csv**  
  Contains attendance records used to calculate total working hours.

- **MotorPHPayroll.iml**  
  IntelliJ IDEA project configuration file.

---

## ⚙️ Features
- Reads employee and attendance data from CSV files  
- Computes total working hours  
- Calculates gross salary  
- Applies deductions (if implemented)  
- Displays payroll details in a structured format  

---

## 🛠️ Tools & Technologies Used
- **Java**
- **IntelliJ IDEA**
- **GitHub**

---

## ▶️ How to Run the Program
1. Open the project in **IntelliJ IDEA**
2. Make sure all CSV files are in the correct directory
3. Run the main Java file
4. View the payroll output in the console
